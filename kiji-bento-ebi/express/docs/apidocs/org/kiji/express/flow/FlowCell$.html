  12. def groupByQualifier[T](cells: Seq[FlowCell[T]]): Seq[(String, Seq[FlowCell[T]])]

    Groups a Seq of flow cells by qualifier.

    Groups a Seq of flow cells by qualifier. The cells must belong to the same column family, and must be in lexicographic sorted order by qualifier (otherwise behavior is undefined). This is the default order for scan results from a Kiji table.

    This method is functionally equivalent to, but more memory efficient than, calling cells.groupBy(_.qualifier).toSeq.

    This method will realize each group into memory individually, so it is safe to iterate through the groups as long as no individual group is larger than the heap. It is NOT safe to hold onto the individual groups while iterating, or call methods that force the seq to fully realize (such as .toList or .toMap) if the provided sequence of flow cells is larger than memory.

    T

    type of datum in cells.

    cells

    to be grouped by qualifier.

    returns

    groups of FlowCells by qualifier.
